NSubset

APIClient. NSubset

Super class for parsed data from API

Constructor

new NSubset(keysnon-null, rawnon-null)

Takes an object and an array of keys to inherit from that object. The keys array can also hold functions, which will be bound to this instance and invoked.

Parameters:
Name Type Description
keys Array.<(String|function())>

Keys to inherit or functions to invoke
raw Object

Any object; it should have members with given keys, of course
